# Proyecto de Automatización de Infraestructura Digital I

Este proyecto consiste en el desarrollo y despliegue de scripts automatizados para la gestión, validación y control de configuraciones en dispositivos e infraestructura de red corporativa.

## Requisitos del Sistema
Para la correcta ejecución y entorno de desarrollo de este proyecto, se requiere contar con el siguiente software instalado:
- Python 3.10 o superior
- Entorno de terminal Git Bash
- Administrador de paquetes pip

## Instrucciones de Instalación
Siga estos pasos en su terminal de Git Bash para clonar e instalar el entorno localmente:

1. Clonar el repositorio:
   ```bash
   git clone [https://github.com/angelamacias-01/parcial2-automatizacion-AM.git](https://github.com/angelamacias-01/parcial2-automatizacion-AM.git)
2. Acceder al directorio raíz del proyecto:
   cd parcial2_automatizacion
